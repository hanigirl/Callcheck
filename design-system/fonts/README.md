# Fonts Directory

הנח את קבצי הפונט Ploni כאן.

## קבצים נדרשים:

הנח את קבצי הפונט `Ploni_ML_v2_AAA` בתיקייה זו:

- `Ploni_ML_v2_AAA.woff2` (מומלץ - הכי קטן ואיכותי)
- `Ploni_ML_v2_AAA.woff` (אופציונלי - fallback)
- `Ploni_ML_v2_AAA.ttf` (אופציונלי - fallback)

## פורמטים:

- **woff2** - הפורמט המודרני ביותר, הכי קטן
- **woff** - fallback לדפדפנים ישנים יותר
- **ttf** - fallback נוסף

אחרי שתניחי את הקבצים כאן, הפונט ייטען אוטומטית דרך `fonts.css`.
